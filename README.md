# More-Fauna-And-Flora
This mod adds 11 animals: 
-Capybaras,
-Pigs,
-Chickens,
-Bulls,
-Parrots,
-Guinea Pigs,
-Llamas,
-Goats,
-Shrimps,
-Octopus,
-Turkeys

And 9 plants:
-Corn,
-Avocados, 
-Apples,
-Pumpkins,
-Watermelons,
-Coconuts,
-Oils,
-Mango,
-Chilis

TileSets for FantasyHex and Hexarealm
